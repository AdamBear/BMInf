# 说明

## 安装方法

### 1. 安装依赖

```
pip install -r requirements.txt
```

### 2. 根据cuda版本安装cupy

#### CUDA 10.1

```
pip install cupy-cuda101
```

#### CUDA 10.2

```
pip install cupy-cuda102
```

### CUDA 11.1

```
pip install cupy-cuda111
```

### CUDA 11.2

```
pip install cupy-cuda112
```

### CUDA 11.3

```
pip install cupy-cuda113
```
